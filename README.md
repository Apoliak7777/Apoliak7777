## 👋 About Me

> Web developer from Slovakia who likes things **clean, minimal and fast**. I build websites front-to-back, lean on JS/PHP only when it earns its place, and care more about clear code than clever code.

```yaml
name:      Apoliak
location:  Slovakia 🇸🇰
role:      Web Developer
focus:     Clean front-end • lightweight back-end • fast load times
philosophy: "Make it work → Make it clear → Make it fast"

approach:
  - Start with a solid HTML/CSS structure
  - Add JavaScript / PHP only when it adds value
  - Keep it minimal, functional and fast

currently_learning:
  - Modular JavaScript architecture
  - Back-end structure & best practices
  - Deployment workflows & CI/CD

collaboration_style:
  - Structured workflows on my own projects
  - Clean forks with clear PR descriptions
  - One concern per pull request
```
